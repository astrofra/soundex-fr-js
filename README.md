Soundex FR Javascript
=================

This script provides a **Soundex FR** integration for a Javascript project. 
Original Soundex FR by [Édouard BERGÉ](http://www.roudoudou.com).

Usage
=====

```es6
Soundex.phonetic("Gendarme") // JANDARM
```
